Veloyd plugin for WordPress WooCommerce

== Description ==

This plugin sends all orders automatically to Veloyd.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-veloyd` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the WooCommerce->Veloyd screen to configure the plugin
